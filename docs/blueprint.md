# **App Name**: ResumePilot

## Core Features:

- File Upload: Allow users to upload resume files (PDF, DOC, HTML) and job description files.
- Keyword Matching and Suitability Analysis: Analyze the uploaded resume and job description to identify matching keywords, generating a 'Is it right for me?' compatibility score.
- Resume Optimization Tool: Generate a new resume tailored to the job requirements, using keyword optimization based on the suitability analysis tool.
- Automated ACT Pass Rate Prediction: Calculate the ACT pass rate after modifying the resume.
- Cover Letter Generation: Generate personalized cover letters based on the optimized resume and job description.
- Follow-up Email Generation: Generate follow-up email drafts tailored to the job applied for, based on the resume and cover letter content.
- User Interface: Display the resume and cover letters using the webapp's UI.

## Style Guidelines:

- Primary color: Deep blue (#224957) for a professional and trustworthy feel.
- Background color: Light gray (#F0F4F8) to ensure readability and a clean interface.
- Accent color: Soft green (#7FB77E) to highlight key information and calls to action.
- Body text: 'Inter', a grotesque sans-serif font known for its readability and modern appearance.
- Headline font: 'Space Grotesk', a sans-serif with a computerized feel.
- Use minimalist and clear icons to represent different sections and functionalities of the app.
- Subtle transitions and animations when generating and displaying content.
